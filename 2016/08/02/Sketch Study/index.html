Sketch 學習筆記

2016年8月2日

###工具介紹
創建畫部(artboard)
建立幾何圖形、編輯幾何圖形、旋轉、圖形工具列、交集聯集、上移下移圖層
群組、解散群組(hotkey介紹)
開啟尺標、開啟及調整背景方格(grid)、開啟及設定layout
圖層介紹

Sketch Shortcuts、Keyboards Shortcuts、9 個你應該知道的 Sketch 功能
(cmd+4: 選取物件至中，alt: hover 顯示距離，ctrl+f: 讓方形填滿artboard，Cmd+Shift+3：擷取全螢幕，Cmd+Shift+4：指定擷取範圍)

文字編輯、複製樣式介紹(調整完一樣式須按 新增的樣式旁重新整理的按鈕)、管理樣式列表(organize text styles)

長寬輸入框可以打算式：例如原本寬度為400px，想減半，可以打400/2，按下enter就可以完成
border-radius也可以很方便設定四邊不同的圓角：在輸入框打10/0/

###Symbol介紹
以header為例：先製作一包含標題及選單的header, 將其全組起來，點選上方”create symbol”按鈕，即可建立起symbol
至左上insert→symbol選擇剛建立的symbol即可置入至頁面
若要編輯已建立的symbol：在symbol上點兩下即可進入
可單獨修改右側屬性質→文字及圖片，不影響其他Symbol
在圖層symbol上點右鍵→Detach from symbol：與Symbol分離，可獨立編輯文字、樣式等。

開啟Sketch內建的Symbol UI：File → New From Template → IOS UI Design
點開後，即可在insert → symbol 中檢視所有內建的symbol UI ( 例如：keyboard, navigation bar…. )
