Sketch 簡介

  • Bohemian Coding 團隊開發 (2010, 荷蘭)
  • Mac Only
  • $99美元 ( 供兩台電腦 )
  • UI Mockup ( Web/App 皆可 )
  • 內建多種版型
  • 可將元件設為Symbol重複利用
  • 輸出方便
  • 資源豐富( 外掛與範本 )
  • 搭配Zeplin、Marvel、Flinto做線上互動




為何使用 Sketch

Sketch 可將設計師想法更完整呈現，使 Designer 與 Developer 溝通更順暢。
